Battery Health Monitoring System using Raspberry Pi (OS modification ) and Modbus TCP.

## Features

- 15 Cell Voltage Monitoring
- Bank Voltage Monitoring
- Current Monitoring
- 15 Temperature Sensors
- SOC Calculation
- SOH Calculation
- Threshold Alerts
- Data Logging
- Web Dashboard

## Hardware

- Raspberry Pi 4
- BHMS Device
- Ethernet Communication

## Network Configuration

BHMS IP : 10.0.0.2  
Raspberry Pi IP : 10.0.0.22  
Port : 5000

## Tools Used

- Raspberry Pi OS
- Python
- pymodbus
- Flask
- CSV Data Storage
